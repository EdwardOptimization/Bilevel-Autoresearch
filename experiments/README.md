# Experiments

Timestamped experiment records. Each subdirectory contains logs, reports, and artifacts from a complete experimental session.

## Naming Convention

`{domain}_{YYYYMMDD}/` — e.g., `train_opt_20260322/`

Each experiment directory should contain:
- `README.md` — experiment setup and summary
- `REPORT.md` — detailed analysis and conclusions
- `logs/` — raw experiment output logs
- `report.json` — structured results data
